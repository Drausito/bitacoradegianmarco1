SEGUNDA SESIÓN

Iniciamos la segunda sesión el 1 de abril, nos tocó una profesora de reemplazo la cual inició comentandonos más a detalle como sería la bitacora que debíamos desarrollar para luego poder trasladar la información más facil y rápido a la página web.

Nuestro trabajo de primer promedio trataría de desarrollar una página web tipo blog, a través de una plantilla donde narraremos lo que aprendamos en el día a día de nuestras clases

después de mostrarnos un ejemplo de Bitácora muy bien desarrollado, la profesora nos envió una plantilla de página web llamada template 7 y nos invitó a descargar el Visual Studio Code

Después de descargar el Visual y la plantilla, debiamos personalizarla, procedimos a examinar cada elemento dentro de la plantilla como son los assets, las imágenes y a saber que es elementos, index y generics

a continuación la profesora nos explicó los pasos básicos para poder reemplazar textos, vectores e imagenes dentro de la plantilla

En lo personal, yo ya había llevado un curso de diseño de interfaces en el ciclo pasado, pero nunca había trabajado encima de una plantilla, además ya ni me acordaba muchas cosas, así que tuve que reaprender a moverme por el Visual, no se me hizo tan complejo y rápidamente pude volver a agarrarle el truco

La profesora nos explicó algunas denominaciones que nos servirá más adelante como:

DENOMINACIONES

Código Para que sirve
html elemento base que abre y cierra una página web
Head Parte no visible de la página web, donde normalmente se nombra a la página, se pone el CSS y los códigos fuente entre otros elementos más que no serán visibles dentro de la página
Tittle Nombre de la página
Body Contenido visible de la página
SRC es la ruta que va a tomar el HTML para buscar imágenes, vectores o fotos
a esta denominación junto con href se usa para generar vínculos hacia otras páginas o a otros html del mismo trabajo
h1, h2, h3, h4, h5, h6 Son encabezados que usamos para categorizar mejor nuestra página
P Denominación para poner los cuerpos de texto
ul,ol,tr,td Denominaciónes para crear tablas ordenadas y desordenadas dentro de la página web

y para finalizar la clase nos enseñó como explorar el código fuente y alterarlo directamente desde la página web presionando F12

esto nos ayudará de forma más facil a poder revisar de forma más gráfica nuestra página web, incluso poder alterar colores para luego replicar la codificación dentro del CSS
