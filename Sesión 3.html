TERCERA SESIÓN

Iniciamos la clase con el profesor revisando los avances de los alumnos, luegos prosiguió a corregir y solucionar dudas de los compañeros, a continuación nos enseñó a como se vinculaba las imagenes y a mantener un orden dentro de las carpetas.

Debemos de procurar siempre escribir bien nuestros vínculos para imagenes como para otras páginas, en caso contrario la página tendrá errores.

En caso de que estemos subiendo una imagen, la cual por las medidas, no calce exacto con las demás, a través de un cógido witdh y height después del link de la imagen, podremos forzar el tamaño de la imagen para que calce en el espacio asignado

Como consejo, debemos de ir probando medidas hasta que encontremos la que encaje perfectamente en el espacio y aplicarle el código a las demás imágenes

Dentro de nuestra plantilla encontraremos un HTML llamado Generics, esta página, es a donde te redirige el index cuando precionas el botón de las secciones y se usará de base para el desarrollo de las secciones futuras.

Para facilitar el flujo de trabajo, deberemos desarrollar correctamente el primer Generics y luego cuando esté terminado, copiarlo las veces que sea necesario y actualizar los códigos de redirección en la página principal del index dentro de cada botón y en el menú el menú se deberá actualizar en cada sección y en el index principal

Para finalizar la clase, el profesor nos dió las indicaciones para la entrega del primer promedio donde nos indicó que debemos tener avanzado las tres primeras sesiones, todo lo que hemos desarrollado en la bitácora lo deberemos trasladar a cada una de las páginas, donde deberemos agregar: Imágenes, hipervinculos y separar cada punto correctamente según se haya visto en la clase
