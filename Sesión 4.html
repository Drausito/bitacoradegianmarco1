CUARTA SESIÓN

Luego de haber desarrollado las primeras tres sesiones, nos tocaba subir la página web a un servidor en línea

Para esto, entraremos a Netlify un servidor donde podremos subir nuestros trabajos de Visual Studio Code y seguiremos los siguientes pasos

PASOS

Como primer paso, entraremos a Netlify y nos loguearemos, al registrarnos por primera ves, nos pedirá verificar nuestro correo electrónico, cuando completemos todos esos pasos, nuestra cuenta estará correctamente creada.
Como segundo paso rellenaremos un cuestionario de quienes somos y para que usaremos el servidor.
La página nos invitará a subir nuestro primer proyecto, a continuación arrastraremos la carpeta que contiene nuestro proyecto de Visual Studio a donde dice or deploy manually y esperaremos a que termine de cargar.
A continuación nos enviará a una página con toda la información actual de nuestra web y le daremos clic al botón change site name donde nos abrirá una opción para nombrar nuestra página no debemos usar ni espacios ni tildes en el nombre.
Para finalizar le daremos clic al link ubicado en la parte de arriba de nuestra página y nos enviará a nuestra página web ya subida en línea.

Para finalizar la clase el profesor nos explicó el proceso que va a venir a continuación en los siguientes promedios y las fases que vamos a tomar para culminar exitosamente nuestro proyecto.

luego de eso el profesor fue pasando por cada grupo para poder ayudarnos a definir el proyecto que desarrollaremos a lo largo del ciclo.
